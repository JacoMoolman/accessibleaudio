# Security policy

## Reporting a vulnerability

Please report security vulnerabilities privately through a
[GitHub security advisory](https://github.com/JacoMoolman/accessibleaudio/security/advisories/new).
Do not include credentials, personal information, exploit details, or other
sensitive evidence in a public issue.

Include the affected URL or component, reproduction steps, impact, and any
suggested remediation. Reports will be acknowledged and assessed as promptly
as practical.

## Scope

The public website and code in this repository are in scope. Do not perform
denial-of-service testing, access another person's data, make a real payment,
or retain data obtained during testing.
